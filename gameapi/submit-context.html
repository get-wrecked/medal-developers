Send Context

Submit game context, including details about the local player, server, and relevant global context tags.

POST http://localhost:12665/api/v1/context/submit

Example Request

HTTP HEADERS
Content-Type application/json
publicKey YOUR_PUBLIC_KEY
REQUEST BODY 
{
    "serverId": "n446av",
    "serverName": "SpaceTurtles RolePlay",
    "localPlayer": {
        "playerId": "trevorP12345",
        "playerName": "Trevor Philips"
    },
    "globalContextTags": {
        "client": "fivem",
        "mode": "survival",
        "server": "strp"
    },
    "globalContextData": {
        "biome": "Plains",
        "weather": "Clear",
        "joinUrl": "https://cfx.re/join/n446av"
    }
}

Request Body Parameters

Property Type Required Description
serverId string No Unique ID of the server
serverName string No Name of the server
localPlayer object No Object containing local player information
customStatus string No Custom activity status to set for the user
globalContextTags object No Tags for global context as hashtags
globalContextData object No Additional metadata for global context
Error Code HTTP Status Description
INVALID_MODEL 400 The request body does not match the expected model structure
MISSING_PUBLIC_KEY 400 The publicKey header is missing from the request
INVALID_APP_DATA 400 Failed to retrieve app data associated with the provided public key
INTERNAL_ERROR 500 An unexpected error occurred while processing the request
